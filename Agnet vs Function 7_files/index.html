Module 2: Agent vs. Function

A Decision Framework

Use this flowchart to decide the right level of autonomy for a task. Start at the top and follow the questions.

The flowchart

  1. Is the task single-step? (One tool call or none) → Use a smart function.
  2. Are the steps predictable? (You know the sequence in advance) → Use a tool-calling function with hardcoded steps.
  3. Does the strategy need to change based on results? (Branching, adaptation) → Consider a bounded agent.
  4. Is there a bounded number of steps? (You can set a reasonable max) → Use a bounded agent with max iterations.
  5. Is the scope truly open-ended? → Use an autonomous agent — but add strong guardrails.

FINRA examples

  • "Summarize this filing" → Smart function (single-step)
  • "Get firm profile, then check violations" → Tool-calling function (predictable steps)
  • "Investigate this firm's compliance posture" → Bounded agent (adapts based on findings)
  • "Conduct a full regulatory examination" → Autonomous agent (open-ended scope)
