Module 2: Agent vs. Function

Exercise: Function or Agent?

Given 4 FINRA business problems, analyze each and implement 2 as working smart functions.

The problems

  1. Filing Summary: Given a regulatory filing ID, retrieve the filing and produce a structured summary with key findings.
  2. Compliance Check: For a given firm, check compliance status across all rule categories and flag any issues.
  3. Cross-Firm Investigation: Given a registered representative, find all firms they're associated with, check each firm's violation history, and identify patterns.
  4. Exam Preparation: Given a firm and an upcoming examination type, gather all relevant data (profile, violations, filings, compliance status) and produce an exam preparation brief.

For each problem

  1. Classify on the autonomy spectrum (1–5)
  2. Justify with cost/complexity/reliability analysis
  3. Implement problems 1 and 2 as smart functions (single-pass tool calling)

Validation

  • Correct classification for at least 3 of 4 problems
  • Both implementations work end-to-end with tool calling
Build it: The sandbox provides the tool definitions from Module 1. Implement your classifications and the two smart functions.
The ability to correctly classify a problem on the autonomy spectrum — and justify your choice — is as important as the ability to implement the solution.
